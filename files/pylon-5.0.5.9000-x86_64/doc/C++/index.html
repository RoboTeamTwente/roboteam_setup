Programmer's Guide and API Reference for pylon for Linux

Introduction

The Basler pylon API is the Basler C++ application programming interface (API) for IEEE 1394, Gigabit Ethernet (GigE), USB and Camera Link cameras. The Linux version of pylon only supports Gigabit Ethernet and USB cameras. The pylon API offers a generic, unified programming interface that hides most of the differences between interface technologies. Basler pylon 5 comes with the next generation of the GenICam API, introducing large improvements in GenICam performance and memory footprint. More detailed information about what's new can be found in the migration guide.

Precautions

Caution: The code snippets provided in this manual are included as sample code only.
