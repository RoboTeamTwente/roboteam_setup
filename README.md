# Archlinux install #

You can install the dependencies on archlinux using the following commands:

* git clone https://github.com/RoboTeamTwente/roboteam_setup.git
* cd roboteam_setup
* chmod 755 ./arch_setup.sh
* ./arch_setup.sh
